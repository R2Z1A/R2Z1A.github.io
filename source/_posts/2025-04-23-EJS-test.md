# EJS 可用性测试

## 1. 输出当前日期
当前时间：<%= new Date().toLocaleString() %>  

## 2. 数学计算
10 + 20 = <%= 10 + 20 %>  

## 3. 访问 Front Matter 变量
自定义变量：<%= page.test_var %>  

## 4. 条件判断
<% if (true) { %>
  条件为真时显示这段文字
<% } else { %>
  条件为假时显示这段文字
<% } %>

## 5. 循环遍历（示例数组）
<% const fruits = ['苹果', '香蕉', '橙子']; %>
水果列表：
<% fruits.forEach(fruit => { %>
- <%= fruit %>
<% }); %>
